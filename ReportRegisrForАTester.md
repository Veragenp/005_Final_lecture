**Перечень используемых инструментов с обоснованием выбора**
1. Selenide - среда для автоматизированного тестирования веб-приложений. В данной задаче требуется протестировать
   web-сайт;
2. Junit5 - выполняем тестирование web-сайта при помощи языка Java, данная среда для тестирования приложений Java;
3. Java-Faker - библиотека для создания тестовых данных: имя, телефон, электронная почта.
4. Gradle - так как в данном проекте используются различные библиотеки и фреймворки, требуется инструмент для сборки проекта.
5. Allure - нужно проанализировать результаты тестирования, для этого используем фреймворк для создания отчетов.

**Перечень необходимых разрешений/данных/доступов**
